ChatRoom with Socket.io and Node.js


How to run application, follow the steps below


Step1: Need installation of NodeJs version 4.3.1 or latest one from site 4.4.0 but you also need to update package.json

Step2: open cmd and run npm install socket.io

Step3: npm install express

Step4: Goto your directory chat2 and run command npm install (Package.json is defined in directory only versions need to edit as per installed version).

Step5: Run this command "node app.js"

Step6: Open Chrome browser and type url http://localhost:3000

Step7: Chat Room is opened you can enter your name and start chatting with other instants.


Finish
